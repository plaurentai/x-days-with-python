## ClydeBank Coffee Shop project

The core requirements of the project include:

Creating a command-line coffee shop application
Displaying menu items and prices
Taking customer orders
Calculating totals and handling basic math operations
Using variables, loops, conditionals, and functions
Managing data using lists, dictionaries, tuples, and sets
Handling user input and validation
Organizing code into reusable modules and functions
Reading and writing data to files
Applying object-oriented programming concepts later in the book

More advanced features include:

inventory management,
order tracking,
error handling,
modular code structure,
and cleaner software design practices.
