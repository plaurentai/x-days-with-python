# X Days of Python Challenge 🚀

Welcome to my **X Days of Python Challenge**!

I am committed to spending at least **one hour every day for the next X days** learning, building, and improving my Python skills.

This journey will focus heavily on:
- Python fundamentals
- Automation
- Artificial Intelligence (AI)
- Investing and trading applications
- Real-world projects

My goal is not only to learn Python, but also to document the journey publicly so others can learn alongside me.

---

## Why X days?

This journey is not limited to 100 days. It is a lifelong commitment to continuous learning, growth, and improvement.

---

## My Rules

1. I will push code and updates to GitHub every day to track my progress publicly.

2. I will maintain an [Activity Log](log.md) documenting what I learned, practiced, and built each day.

3. I will focus on consistency over perfection.

4. I will learn by building real-world projects and experimenting with ideas.

5. I will share lessons, mistakes, and insights throughout the journey.
